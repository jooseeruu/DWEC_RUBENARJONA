Browser Object Model (BOM) - Reference Guide

El BOM permite interactuar con el navegador mediante una serie de objetos y métodos que permiten controlar aspectos de la ventana, la ubicación, la navegación, entre otros.

1. Window Object

El objeto window es el objeto global del BOM y contiene métodos y propiedades fundamentales.

  • window.open(): Abre una nueva ventana o pestaña.
  • window.close(): Cierra la ventana actual.
  • window.alert(): Muestra un cuadro de alerta.
  • window.confirm(): Muestra un cuadro de confirmación con opciones Aceptar/Cancelar.
  • window.prompt(): Solicita entrada de texto del usuario.

Documentación: MDN - Window

2. Location Object

El objeto location contiene información sobre la URL actual de la ventana.

  • window.location.href: Obtiene o establece la URL completa.
  • window.location.reload(): Recarga la página actual.
  • window.location.assign(): Carga un nuevo documento.
  • window.location.replace(): Reemplaza el documento actual con una nueva URL sin guardar el estado en el historial.

Documentación: MDN - Location

3. History Object

El objeto history permite controlar el historial de navegación del usuario.

  • window.history.back(): Navega una página atrás en el historial.
  • window.history.forward(): Navega una página adelante en el historial.
  • window.history.go(): Navega un número específico de páginas en el historial.
  • window.history.length: Devuelve el número de entradas en el historial.

Documentación: MDN - History

4. Navigator Object

El objeto navigator proporciona información sobre el navegador del usuario.

  • navigator.userAgent: Proporciona información sobre el navegador y el sistema operativo.
  • navigator.language: Devuelve el idioma preferido del navegador.
  • navigator.geolocation: Accede a la ubicación geográfica (requiere permisos).

Documentación: MDN - Navigator

5. Screen Object

El objeto screen proporciona información sobre la pantalla del dispositivo.

  • screen.width: Ancho de la pantalla en píxeles.
  • screen.height: Alto de la pantalla en píxeles.
  • screen.availWidth: Ancho disponible (excluyendo elementos como la barra de tareas).
  • screen.availHeight: Alto disponible.

Documentación: MDN - Screen
